Актуальная инструкция находится в README.md.

Быстрый запуск:
1. Запустите Docker Desktop и Ollama.
2. Выполните: ollama pull qwen2.5:7b-instruct-q4_K_M
3. Запустите launch.bat.
4. Откройте http://localhost:8501

Данные новой версии сохраняются в папке data.
Старый индекс faiss_index оставлен как резервная копия.
